# 🚀 QuantumIDE

QuantumIDE — это высокотехнологичная мобильная среда разработки (IDE) нового поколения со встроенным искусственным интеллектом для веб-разработки, работающая нативно на Android и построенная на базе Flutter. Она предоставляет полноценное окружение разработчика: редактор, терминал, изолированную песочницу, интеграцию с Git и интеллектуального ассистента.

---

## 🛠️ Стек технологий

- **Фреймворк**: [Flutter](https://flutter.dev) (интерфейс Material 3, адаптивная разметка)
- **Управление состоянием**: [Riverpod](https://riverpod.dev)
- **Маршрутизация**: [GoRouter](https://pub.dev/packages/go_router)
- **Ядро редактора**: `re_editor` + `re_highlight`
- **Эмулятор терминала**: `xterm.dart` + `flutter_pty`
- **ИИ-ядро**: `google_generative_ai` (Gemini) + кастомные коннекторы к DeepSeek, Groq, OpenRouter и локальным ИИ (Ollama/LM Studio)
- **Темизация и дизайн**: `flex_color_scheme` + визуальные эффекты матового стекла (glassmorphism)
- **Типографика**: *Outfit* (интерфейс) и *JetBrains Mono* (для кода)

## 📂 Структура проекта

Ниже приведена структура каталогов исходного кода приложения (`lib/`):

*   **`lib/main.dart`**: Главная точка входа Flutter-приложения.
*   **`lib/app.dart`**: Настройки `MaterialApp`, тем оформления и роутера.
*   **`lib/core/`**: Системное ядро и глобальное состояние:
    *   `models/`: Системные структуры данных (диагностика кода, конфигурация провайдеров ИИ).
    *   `providers/`: Глобальные провайдеры состояния (например, локализация).
    *   `router/`: Настройка путей GoRouter.
    *   `theme/`: Конфигурация тем FlexColorScheme и параметров размытия.
    *   `utils/`: Утилиты, конвертер путей (хост-система <-> песочница).
    *   `services/`: Фоновые сервисы и бизнес-логика:
        *   `ai_service.dart`, `ai_autocomplete_service.dart`, `ai_context_compressor.dart`: Интеграция ИИ, автодополнение с дебаунсом и сжатие контекста.
        *   `lsp_service.dart`, `lsp_autocomplete_service.dart`, `analysis_service.dart`: Интеграция протокола LSP, фоновый анализатор Dart и автодополнение.
        *   `git_service.dart`, `diff_service.dart`: Управление git-командами и подсветка изменений на полях (gutter).
        *   `runtime_service.dart`, `native_terminal.dart`: Развёртывание PRoot окружения Ubuntu и мост к локальным процессам shell.
        *   `package_service.dart`, `pub_package_service.dart`: Управление SDK, APT-пакетами, сборками Gradle и pub.dev API.
        *   `workspace_service.dart`, `project_service.dart`: Управление рабочими областями, проектами и шаблонами сборки.
        *   *В разработке*: `ai_agent_orchestrator.dart`, `crdt_sync_service.dart`, `microvm_service.dart`, `wasm_plugin_runner.dart`.
*   **`lib/features/`**: Функциональные модули приложения (Feature-Driven Architecture):
    *   `ai_assistant/`: Панель чата ИИ в реальном времени, настройки и интеграция серверов MCP.
    *   `editor/`: Редактор кода, менеджер вкладок, панель ошибок, структура кода и панель виртуальных клавиш.
    *   `file_explorer/`: Избранное, дерево файлов, глобальный поиск по проекту и анализатор диска.
    *   `git/`: Панель управления версиями, просмотр разниц (diff) и страница разрешения конфликтов.
    *   `home/`: Рабочий стол, создание проектов по шаблонам, панель локальных серверов и настройки IDE.
    *   `preview/`: Веб-превью на базе WebView с просмотром консольных логов и управлением браузером.
    *   `terminal/`: Многовкладочный терминал, панель виртуальных клавиш, сборщик APK и подпись приложений.
    *   *В разработке*: `collaboration/` (Live Share), `plugins/` (менеджер WASM плагинов).
*   **`lib/models/`**: Общие модели данных (сессии чата, пакеты).
*   **`lib/shared/`**: Общие компоненты UI:
    *   `layout/`: Разделитель окон для Fold-устройств (`foldable_split_manager.dart`).
    *   `widgets/`: Стеклянные контейнеры, хлебные крошки, статус-бар и базовые сетки UI.
*   **`lib/l10n/`**: Файлы локализации (RU/EN).

## 🟢 Реализованный функционал

- **Менеджер рабочих областей**: Выбор папок через Android Storage Access Framework (SAF), закладки, шаблоны проектов, список недавних файлов.
- **Редактор Quantum Editor**: Виртуализация строк, подсветка синтаксиса для многих языков, автосохранение, вкладки, индикатор изменений в файлах, линтер и кастомная панель быстрых символов.
- **Встроенный терминал и песочница**: Терминал PTY на базе Ubuntu 24.04 PRoot. Автоматически монтирует накопители Android, поддерживает автозапуск, автовосстановление процессов и имеет кастомный `.bashrc`.
- **ИИ-Ассистент (Мультимодельный)**: Встроенный чат с поддержкой Google Gemini, Groq, DeepSeek, OpenRouter и локальных моделей (Ollama/LM Studio). Интеллектуальное автодополнение кода на лету с дебаунсом в 700 мс.
- **Интеграция с Git**: Просмотр измененных файлов, коммиты, история разницы (Diff) и графическое разрешение конфликтов слияния.
- **Инструменты сборки и диагностики для Android**: Автоматическая настройка NDK/SDK под ARM64 Linux, сборка APK, подпись ключами, интеграция с JDK 17, запуск `dart analyze` в фоновом режиме, панель отображения системных ошибок.
- **Браузерное превью**: Встроенный WebView с инспектором консоли и мониторингом запущенных локальных веб-серверов.
- **Премиальный UI/UX**: Эффекты Glassmorphism (размытие BackdropFilter), плавные переходы и пружинные анимации, выбор акцентных цветов проекта, шрифты Outfit и JetBrains Mono, кастомные иконки Lucide.
- **Локализация**: Полный перевод интерфейса на русский и английский языки.

## 🔮 Дорожная карта (План развития)

Наши цели по разработке новых модулей:

- [ ] **Интеграция LSP**: Полноценная поддержка протокола языкового сервера для HTML, CSS, JavaScript и TypeScript внутри песочницы для автодополнения (по сигнатурам), перехода к объявлению и глубокого линтинга.
- [ ] **Совместная разработка (Live Share)**: Совместное редактирование кода в реальном времени с синхронизацией курсоров с помощью CRDT.
- [ ] **Система плагинов WASM**: Возможность загрузки и выполнения плагинов, написанных на Rust, Go или C++, скомпилированных в WebAssembly для расширения возможностей редактора.
- [ ] **Виртуализация KVM (MicroVM)**: Поддержка аппаратной виртуализации Linux на Android 15+ для обхода ограничений PRoot.
- [ ] **Поддержка гибких экранов**: Интеллектуальное разделение панелей (терминал, редактор, превью) при сгибе складных смартфонов и планшетов.
- [ ] **Локальный автокомплит (MLC LLM)**: Интеграция легковесных моделей (например, Qwen-Coder) для работы автодополнения на NPU чипе устройства без интернета.
